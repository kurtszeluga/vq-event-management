GoDaddy Event Feed Demo

This is a draft embeddable version of the public events listing. If this looks right, we can use the same feed on a GoDaddy test page without finishing the whole registration workflow first.

Paste this into a GoDaddy HTML block: 
<div
  data-vq-feed
  data-category="events"
  data-source-url="https://vq-event-management.vercel.app/api/public-events"
></div>
<script src="https://vq-event-management.vercel.app/godaddy-event-feed.js" defer></script>
